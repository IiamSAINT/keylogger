# Keylogger using `pynput`

## Description

This is a simple **Python-based keylogger** that captures and logs keystrokes from the user's keyboard using the `pynput` library. Every key pressed is saved to a file named `logFile.txt` in the current working directory.

> ⚠️ **Ethical Warning**  
> This program is intended for **educational and ethical purposes only**. Do not use it without the **explicit permission** of the device owner. Unauthorized use may violate laws and privacy rights.

---

## Features

- Logs all keystrokes in real-time
- Saves the keystrokes to a local file (`logFile.txt`)
- Displays each key pressed in the terminal


---

## Prerequisites

- Python 3.x
- `pynput` library

You can install the `pynput` library via pip:

```bash
pip install pynput
